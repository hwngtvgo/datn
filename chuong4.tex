\section{Thiết kế kiến trúc}
\subsection{Lựa chọn kiến trúc phần mềm}

Hệ thống học TOEIC trực tuyến được thiết kế dựa trên kiến trúc Client-Server kết hợp với mô hình MVC (Model-View-Controller) và REST API. Kiến trúc này được chọn vì những ưu điểm sau:

\textbf{Kiến trúc 3-lớp (Three-tier Architecture):}
\begin{itemize}
    \item \textbf{Presentation Layer (Tầng trình bày):} Sử dụng React.js với TypeScript và Tailwind CSS để xây dựng giao diện người dùng tương tác cao, responsive design phù hợp với nhiều thiết bị
    \item \textbf{Business Logic Layer (Tầng logic nghiệp vụ):} Sử dụng Spring Boot với Java để xử lý các quy tắc nghiệp vụ, authentication/authorization, và điều phối hoạt động giữa tầng trình bày và tầng dữ liệu
    \item \textbf{Data Layer (Tầng dữ liệu):} Sử dụng MySQL để lưu trữ dữ liệu bền vững và Spring Data JPA để thao tác với cơ sở dữ liệu
\end{itemize}

\textbf{Áp dụng vào hệ thống TOEIC cụ thể:}

\textbf{Frontend (React + TypeScript):}
\begin{itemize}
    \item \textbf{Pages:} Quản lý các trang chính như HomePage, Learning, PracticeTests, Admin Dashboard với routing thông qua React Router DOM
    \item \textbf{Components:} Các component tái sử dụng sử dụng Shadcn/UI và Radix UI primitives như Card, Button, Dialog, Table
    \item \textbf{Services:} Các service để giao tiếp với REST API backend thông qua Axios (authService, testResultService, userService)
    \item \textbf{Context:} AuthContext để quản lý state authentication toàn cục
\end{itemize}

\textbf{Backend (Spring Boot):}
\begin{itemize}
    \item \textbf{Controllers:} UserController, TestController, ToeicQuestionController, AuthController xử lý HTTP requests và trả về responses
    \item \textbf{Services:} UserService, ToeicTestService, TestResultService, AuthService chứa business logic
    \item \textbf{Repositories:} UserRepository, TestRepository, ToeicQuestionRepository sử dụng Spring Data JPA để truy cập dữ liệu
    \item \textbf{Security:} Spring Security với JWT authentication và role-based authorization
    \item \textbf{FileController:} File upload/delete với Cloudinary
\end{itemize}

\textbf{Database (MySQL):}
\begin{itemize}
    \item Lưu trữ thông tin users, roles, tests, questions, test\_results
    \item Sử dụng JPA entities để mapping với database tables
    \item Foreign key constraints đảm bảo data integrity
\end{itemize}

Kiến trúc này cho phép phát triển song song frontend và backend, dễ dàng scale và maintain. Việc tách biệt rõ ràng giữa các tầng giúp tăng tính modular và khả năng test.

\subsection{Thiết kế tổng quan}

Hệ thống được tổ chức thành các package theo kiến trúc phân tầng rõ ràng:

\textbf{Frontend Packages:}
\begin{itemize}
    \item \textbf{/src/pages:} Chứa các component trang chính
    \begin{itemize}
        \item \texttt{Home.tsx} - Trang chủ hiển thị đề thi mới nhất
        \item \texttt{Learning/} - Các trang học tập theo level (0-300, 300-600, 600-800)
        \item \texttt{PracticeTests/} - Trang luyện đề với TestPage.tsx
        \item \texttt{Admin/} - Admin Dashboard, Users, ToeicExams, ToeicQuestions
        \item \texttt{User/} - TestHistory, TestResultDetail, TestStatistics
    \end{itemize}
    \item \textbf{/src/components:} UI components tái sử dụng
    \begin{itemize}
        \item \texttt{ui/} - Shadcn/UI components (Button, Card, Dialog, Table)
        \item \texttt{admin/} - AdminSidebar cho navigation
        \item \texttt{toeic/} - ExamForm cho quản lý đề thi
        \item \texttt{Layout.tsx} - Layout wrapper chung
    \end{itemize}
    \item \textbf{/src/services:} API communication services
    \item \textbf{/src/contexts:} React contexts (AuthContext)
    \item \textbf{/src/types:} TypeScript type definitions
\end{itemize}

\textbf{Backend Packages:}
\begin{itemize}
    \item \textbf{controllers:} REST API endpoints
    \begin{itemize}
        \item \texttt{UserController} - User management (/api/users)
        \item \texttt{AuthController} - Authentication (/api/auth)
        \item \texttt{TestController} - Test management (/api/tests)
        \item \texttt{ToeicQuestionController} - Question management (/api/toeic-questions)
        \item \texttt{TestResultController} - Test results (/api/test-results)
        \item \texttt{FileController} - File upload/delete với Cloudinary (/api/files)
    \end{itemize}
    \item \textbf{services:} Business logic layer
    \begin{itemize}
        \item \texttt{CloudinaryService} - Quản lý file upload/delete trên Cloudinary
    \end{itemize}
    \item \textbf{repositories:} Data access layer with Spring Data JPA
    \item \textbf{models:} JPA entities (User, Test, ToeicQuestion, TestResult)
    \item \textbf{security:} JWT authentication và authorization
    \item \textbf{payload:} Request/Response DTOs
\end{itemize}

\subsection{Thiết kế chi tiết gói}

    \item \textbf{Repository:} UserRepository extends JpaRepository<User, Long>

\textbf{Gói Test Management:}
\begin{itemize}

\textbf{Gói Test Results:}
\begin{itemize}

    \item \textbf{Repository:} TestResultRepository, UserAnswerRepository
    \item \textbf{Entities:} TestResult, UserAnswer với detailed scoring fields
\end{itemize}

\subsection{Thiết kế lớp}

\textbf{Lớp User:}
\begin{itemize}
    \item \textbf{Thuộc tính:} 
    \begin{itemize}
        \item id: Long (Primary Key)
        \item username: String (Unique, Not Null)
        \item email: String (Unique, Not Null) 
        \item password: String (Encrypted)
        \item fullName: String
        \item roles: Set<Role> (ManyToMany)
        \item active: boolean (Default true)
        \item createdAt: LocalDateTime
        \item updatedAt: LocalDateTime
    \end{itemize}
    \item \textbf{Phương thức:} 
    \begin{itemize}
        \item User() - Default constructor
        \item User(username, password, email, fullName) - Constructor
        \item Getters/Setters cho tất cả fields
        \item equals(), hashCode(), toString()
    \end{itemize}
    \item \textbf{Quan hệ:} 
    \begin{itemize}
        \item @ManyToMany với Role (user\_roles table)
        \item OneToMany với TestResult
        \item OneToOne với NotificationSetting
    \end{itemize}
\end{itemize}

\textbf{Lớp Test:}
\begin{itemize}
    \item \textbf{Thuộc tính:}
    \begin{itemize}
        \item id: Long (Primary Key)
        \item title: String (Not Null)
        \item description: String (TEXT)
        \item type: TestType (Enum: FULL, MINI, PRACTICE)
        \item duration: Integer (phút)
        \item difficulty: DifficultyLevel (EASY, MEDIUM, HARD)
        \item isActive: Boolean
        \item createdAt: LocalDateTime
        \item createdBy: String
        \item questionGroups: List<QuestionGroup> (ManyToMany)
    \end{itemize}
    \item \textbf{Phương thức:}
    \begin{itemize}
        \item Test() - Default constructor
        \item addQuestionGroup(QuestionGroup)
        \item removeQuestionGroup(QuestionGroup)
        \item getTotalQuestions(): Integer
        \item Getters/Setters
    \end{itemize}
    \item \textbf{Quan hệ:}
    \begin{itemize}
        \item @ManyToMany với QuestionGroup (test\_question\_groups table)
        \item OneToMany với TestResult
    \end{itemize}
\end{itemize}

\textbf{Lớp ToeicQuestion:}
\begin{itemize}
    \item \textbf{Thuộc tính:}
    \begin{itemize}
        \item id: Long (Primary Key)
        \item question: String (TEXT, Not Null)
        \item questionOrder: Integer
        \item correctAnswer: String
        \item explanation: String (TEXT)
        \item difficultyLevel: DifficultyLevel (Enum)
        \item category: QuestionCategory (GRAMMAR, VOCABULARY, LISTENING)
        \item questionGroup: QuestionGroup (ManyToOne, nullable)
        \item options: Set<ToeicOption> (OneToMany)
    \end{itemize}
    \item \textbf{Phương thức:}
    \begin{itemize}
        \item ToeicQuestion() - Default constructor
        \item addOption(ToeicOption)
        \item removeOption(ToeicOption)
        \item isCorrectAnswer(String): boolean
        \item getCorrectOption(): ToeicOption
        \item Getters/Setters
    \end{itemize}
    \item \textbf{Quan hệ:}
    \begin{itemize}
        \item @ManyToOne với QuestionGroup (nullable)
        \item @OneToMany với ToeicOption (cascade)
        \item OneToMany với UserAnswer
    \end{itemize}
\end{itemize}

\textbf{Lớp TestResult:}
\begin{itemize}
    \item \textbf{Thuộc tính:}
    \begin{itemize}
        \item id: Long (Primary Key)
        \item user: User (ManyToOne, Not Null)
        \item test: Test (ManyToOne, Not Null)
        \item listeningScore: Integer
        \item readingScore: Integer
        \item grammarScore: Integer
        \item vocabularyScore: Integer
        \item totalScore: Integer
        \item listeningScaledScore: Integer (5-495 range)
        \item readingScaledScore: Integer (5-495 range)
        \item completionTimeInMinutes: Integer
        \item correctAnswers: Integer
        \item totalQuestions: Integer
        \item createdAt: LocalDateTime
        \item answers: List<UserAnswer> (OneToMany)
    \end{itemize}
    \item \textbf{Phương thức:}
    \begin{itemize}
        \item TestResult() - Default constructor
        \item calculateScore(): void
        \item getAccuracyRate(): Double
        \item isCompleted(): boolean
        \item getScaledScore(): Integer
        \item Getters/Setters
    \end{itemize}
    \item \textbf{Quan hệ:}
    \begin{itemize}
        \item @ManyToOne với User
        \item @ManyToOne với Test  
        \item @OneToMany với UserAnswer (cascade)
    \end{itemize}
\end{itemize}

\subsection{Thiết kế cơ sở dữ liệu}

\textbf{Entity-Relationship Diagram:}

Cơ sở dữ liệu được thiết kế với các entity chính và relationships:

\textbf{Bảng users:}
\begin{itemize}
    \item id (BIGINT, PK, AUTO\_INCREMENT)
    \item username (VARCHAR(50), UNIQUE, NOT NULL)
    \item email (VARCHAR(255), UNIQUE, NOT NULL)
    \item password (VARCHAR(255), NOT NULL) - BCrypt hashed
    \item full\_name (VARCHAR(255))
    \item active (BOOLEAN, DEFAULT TRUE)
    \item verification\_token (VARCHAR(255))
    \item token\_expiry\_date (DATETIME)
    \item created\_at (DATETIME, NOT NULL)
    \item updated\_at (DATETIME)
\end{itemize}

\textbf{Bảng roles:}
\begin{itemize}
    \item id (INT, PK, AUTO\_INCREMENT)
    \item name (VARCHAR(20), ENUM: 'ROLE\_USER', 'ROLE\_ADMIN')
\end{itemize}

\textbf{Bảng user\_roles:}
\begin{itemize}
    \item user\_id (BIGINT, FK -> users.id)
    \item role\_id (INT, FK -> roles.id)
    \item PRIMARY KEY (user\_id, role\_id)
\end{itemize}

\textbf{Bảng tests:}
\begin{itemize}
    \item id (BIGINT, PK, AUTO\_INCREMENT)
    \item title (VARCHAR(255), NOT NULL)
    \item description (TEXT)
    \item type (VARCHAR(20), ENUM: 'FULL', 'MINI', 'PRACTICE')
    \item duration (INT, NOT NULL) - thời gian tính bằng phút
    \item difficulty (VARCHAR(10), DEFAULT 'MEDIUM')
    \item instructions (TEXT)
    \item is\_active (BOOLEAN, DEFAULT TRUE)
    \item created\_at (DATETIME)
    \item created\_by (VARCHAR(255))
\end{itemize}

\textbf{Bảng question\_groups:}
\begin{itemize}
    \item id (BIGINT, PK, AUTO\_INCREMENT)
    \item title (VARCHAR(255), NOT NULL)
    \item question\_type (VARCHAR(20), ENUM: 'LISTENING', 'READING')
    \item part (INT) - Part 1-7 theo chuẩn TOEIC
    \item passage (TEXT) - Đoạn văn cho Reading questions
    \item audio\_url (VARCHAR(500)) - Link file audio cho Listening
    \item image\_url (VARCHAR(500)) - Link hình ảnh
    \item test\_id (BIGINT, FK -> tests.id)
\end{itemize}

\textbf{Bảng toeic\_questions:}
\begin{itemize}
    \item id (BIGINT, PK, AUTO\_INCREMENT)
    \item question (TEXT, NOT NULL)
    \item question\_order (INT)
    \item correct\_answer (VARCHAR(1)) - A, B, C, D
    \item explanation (TEXT)
    \item difficulty\_level (VARCHAR(10), ENUM: 'EASY', 'MEDIUM', 'HARD')
    \item category (VARCHAR(20), ENUM: 'GRAMMAR', 'VOCABULARY', 'LISTENING')
    \item question\_group\_id (BIGINT, FK -> question\_groups.id, NULLABLE)
\end{itemize}

\textbf{Bảng toeic\_options:}
\begin{itemize}
    \item id (BIGINT, PK, AUTO\_INCREMENT)
    \item question\_id (BIGINT, FK -> toeic\_questions.id)
    \item option\_key (VARCHAR(1)) - A, B, C, D
    \item option\_text (TEXT, NOT NULL)
    \item option\_order (INT)
\end{itemize}

\textbf{Bảng test\_results:}
\begin{itemize}
    \item id (BIGINT, PK, AUTO\_INCREMENT)
    \item user\_id (BIGINT, FK -> users.id, NOT NULL)
    \item test\_id (BIGINT, FK -> tests.id, NOT NULL)
    \item listening\_score (INT)
    \item reading\_score (INT)
    \item grammar\_score (INT)
    \item vocabulary\_score (INT)
    \item total\_score (INT)
    \item listening\_scaled\_score (INT) - Điểm chuẩn TOEIC 5-495
    \item reading\_scaled\_score (INT) - Điểm chuẩn TOEIC 5-495
    \item completion\_time\_in\_minutes (INT)
    \item correct\_answers (INT)
    \item total\_questions (INT)
    \item created\_at (DATETIME, NOT NULL)
\end{itemize}

\textbf{Bảng user\_answers:}
\begin{itemize}
    \item id (BIGINT, PK, AUTO\_INCREMENT)
    \item test\_result\_id (BIGINT, FK -> test\_results.id, NOT NULL)
    \item question\_id (BIGINT, FK -> toeic\_questions.id, NOT NULL)
    \item selected\_answer (VARCHAR(1)) - A, B, C, D
    \item is\_correct (BOOLEAN)
    \item answered\_at (DATETIME)
\end{itemize}

\textbf{Relationships và Constraints:}
\begin{itemize}
    \item users 1:N test\_results (Một user có nhiều kết quả thi)
    \item tests 1:N question\_groups (Một test có nhiều nhóm câu hỏi)
    \item question\_groups 1:N toeic\_questions (Một nhóm có nhiều câu hỏi)
    \item toeic\_questions 1:N toeic\_options (Một câu hỏi có nhiều lựa chọn)
    \item test\_results 1:N user\_answers (Một kết quả có nhiều câu trả lời)
    \item Foreign key constraints với ON DELETE CASCADE cho data integrity
    \item Indexes trên foreign keys và search fields
\end{itemize}

\subsection{Minh họa các chức năng chính}

\textbf{Trang chủ (HomePage):}
Giao diện chính với hero section, features cards, và grid hiển thị 3 đề thi mới nhất. Sử dụng responsive design với Tailwind CSS, loading skeletons khi fetch data từ API.

\textbf{Trang luyện đề (PracticeTests):}
Hiển thị danh sách đề thi với filtering và pagination. Mỗi đề thi có card hiển thị thông tin cơ bản, difficulty level, và thời gian làm bài.

\textbf{Trang làm bài (TestPage):}
Interface tối ưu cho việc làm bài với countdown timer, question navigation, auto-save mechanism. Hỗ trợ audio player cho listening questions và responsive design cho mobile.

\textbf{Admin Dashboard:}
Comprehensive management interface với statistics cards, user management table, exam creation forms. Sử dụng charts để hiển thị thống kê và real-time data updates.

\textbf{Trang kết quả (TestResultDetail):}
Detailed analysis với breakdown theo từng skill, score comparison, progress charts, và recommendations cho improvement.

\textbf{Trang học tập (Learning):}
Level-based learning paths (0-300, 300-600, 600-800) với module cards cho Vocabulary, Grammar, và Practice Exams. Mỗi module có progress tracking.
